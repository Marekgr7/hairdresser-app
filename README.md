

#  Haidresser Application

## Our Goal

Our goal is to take our programming skills to next level using microservices.



## What we have created

Hairdresser application with diffrent roles

- admin
- employee
- customer

## What we have used

*   Java 11
*   Spring Boot >2
*   OAuth2
*   Eureka Naming Server
*   RabbitMQ
*   Zipkin
*   Zuul API Gateway
*   Load Balancing
*   Distributed tracing
